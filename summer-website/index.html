暑訓簡介

認識 IZCC 四校聯合暑訓

IZCC四校聯合暑訓

由建中資訊（INFOR）、中山資研（ZSISC）、成功電研（CKCSC）、景美電資（CMIOC）舉辦一年一度的大型活動！參加四校聯合暑訓會是你上高中前最不後悔的決定！在這七天中你可以認識更多升高中的朋友來拓展你的交友圈，也會在活動當中更熟悉高中生活，認識更多學長姊。

同時又可以學習更多關於Python、C++、tkinter、網頁設計的知識，還會參觀資訊產業的公司，也當然會讓大家遊玩各式各樣好玩的神秘活動，期待你們的參與！

活動介紹

我們在7天中的6天都安排了有趣的不同活動

Day 1 - 大地遊戲

大地遊戲又名曠野遊戲、越野遊戲。在這次的暑訓中，我們會於中正紀念堂由玩大地遊戲，過程中會有許多好玩、刺激的關卡，每一小隊會在大地遊戲中盡全力在每一關中拿取分數，並且在最後爭取最高名次！

Day 2 - 企業參訪

以後想成為工程師嗎？那你絕對不能錯過這項活動！我們即將前往「精誠資訊」，精誠資訊 SYSTEX Corporation 成立於 1997 年，是台灣資訊服務產業龍頭企業，擁有近 3,000 名員工，位居台灣前一百大服務業。我們將邀請企業的員工，來為我們講解工程師的生活。

Day 3 - RPG

只有玩過線上版的 RPG 嗎?那你一定要來體驗看真人版 RPG。有著刺激且燒腦的劇情跟關卡等你來破解，至於劇情是什麼，等你來就知道拉!

Day 4 - 大型團康活動

在日常生活中，我們常常因為生活過於忙碌而忘了關心周遭。而在這個活動中，我們將帶領更加認識雙北，敬請期待! 詳細活動內容，等你來就知道了！

Day 5 - 趣味遊戲

將搭配著許多桌遊、神秘小遊戲，在一次次精彩的奮鬥與交戰中，成為最終的贏家！

Day 7 - 神秘活動

為此營隊中最精彩的活動，會讓你們有個難忘的夜晚，知道是什麼就不好玩啦，敬請期待！

課程介紹

我們有許多不同的資訊相關課程，一起來認識這些課程吧！

學習新知

在營隊中，你不光可以參加有趣的活動，更可以學到扎實的學術知識！我們將以淺顯易懂的方式讓即使是零基礎也可以學會！

  • 01 C++

    歡迎來到基礎C++課程！在這裡，你將踏上學習C++的旅程，掌握其核心概念和實用技巧。我們將深入淺出的學習以下主題，讓你能夠建立堅實的程式設計基礎。課程內容包含：

    - 編寫C++的電腦環境配置

    - 輸入/輸出

    - 各種變數型態與其應用

    - if else判斷式

    - 各類運算子

    - for while迴圈

  • 02 演算法

    本課程將以理論結合實踐，幫助學員掌握演算法的基本概念和應用技巧，為後續的高階學習打下堅實的基礎。 演算法課程內容涵蓋以下主題：

    - 何謂演算法

    - 演算法複雜度分析

    - 基礎遞迴觀念

    - 基礎演算法與例題

  • 03 HTML & CSS

    呀勒呀勒，大小姐 is time to寫網頁嘍—想了解HTML網頁的基本架構嗎？本次的課程會向各位大小姐們介紹網頁的基礎-HTML，HTML是一種標記語言常用於建立網頁。我們還會深入的介紹HTML的基本語法。什麼？大小姐擔心從來都沒學過？沒關係，我們的課程是從0開始！0基礎也沒問題！準備好了嗎大小姐？課程內容包含：

    - 網頁的結構

    - 網頁的常用標籤(連接/表單/結構元素)

    - 階層樣式表 CSS

    - CSS 選擇器

  • 04 Discord 機器人

    Discord 是一個高度自定義的社群軟體，課程中會帶你使用Python一步步完成自己的機器人，讓你能夠將Discord內的各種操作自動化！以下是課程會學到的內容：

    - Discord的基本介紹

    - Discord 機器人的原理

    - 身份組、頻道、成員、表符、伺服器等等的管理。

    - 設計自定義的酷酷指令

    - 在自己的網站中用 Discord 登入

    - 更多內容敬請期待！

  • 05 Python

    Python 是一款非常適合初學者的程式語言，因其語意簡單直白的特性，在行業中非常受歡迎，如現在最流行的AI，程式大部分都是由Python撰寫，在這個課程中，將會由淺入深，帶著你學會課程內容包含：

    - Python的基礎語法

    - 程式語言的運作方式

    - Python進階用法

    - 專案實作

  • 06 Tkinter

    Tkinter 是Python中標準的GUI（圖形用戶界面）工具包，可用於製作小遊戲或創建簡易的應用工具。透過Tkinter，你可以輕鬆的做出視覺化界面，無需了解複雜的圖形程式設計技巧，在這課程你可以學到:

    - 建立視窗界面

    - 運用元件製作出按鈕、輸入框等功能

    - 佈局管理

    - 事件驅動等相關內容

社團介紹

IZCC 四校聯合暑訓是由四校的四個社團一起舉辦的，來認識它們吧！

INFOR 37th

一建國中學資訊社37屆一

建中資訊社是一個以電腦科技為主題的社團，歡迎對程式設計、網頁製作、遊戲開發、人工智慧等領域有興趣的同學加入。在資訊社，你可以學習到最新的資訊技術，並且與其他社員一起分享作品與想法。

CKCSC 37th

一成功高中電子計算機研習社37屆一

「成功高中電子計算機研習社 Cheng Gong Computer Study Club」簡稱成功電研CKCSC，是一個成功高中內的學術性社團，在社課會教一些關於資訊領域的課程，會有學長帶你從0開始學，也會有進階班，歡迎學弟加入電研社的大家庭！ 除了教學課程，我們也與IZCC分別是建中資訊、中山資研、景美電資等三校，一同合辦許多活動與聯課，並且也會在放學後與四校共同開設課程，提供四校學弟妹與友校們交流的機會。

ZSISC 31st

一中山女高資訊研究社31屆一

中山資研，又稱楓資，全名為中山女高資訊研究社，社課主要以教學C++為主，無論是程式新手或是電神，都可以跟幹部們一起學習，我們也會分享有關於資訊方面的事情，在放學後也有自由選擇是否參加的不同課程，同時我們也有許多與友校交流的活動，只要對於資訊方面有興趣的學妹，或是想嘗試看看的，都歡迎加入楓資！

CMIOC 31st

一景美女中電腦資訊社31屆一

歡迎來到景美女中的電腦資訊社！我們是一個致力於推廣科技與計算機知識的社團，擁有豐富的學習與實踐機會。不論你是對編程充滿熱情的新手，還是想進一步探索電腦科學領域的高手，我們都歡迎你加入我們的行列。
