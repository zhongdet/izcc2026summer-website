IZCC 四校聯合暑訓

由建中資訊（INFOR）、成功電研（CKCSC）、中山資研（ZSISC）、景美電資（CMIOC）舉辦一年一度的大型活動！參加四校聯合暑訓會是你上高中前最不後悔的決定！在這七天中你可以認識更多升高中的朋友來拓展你的交友圈，也會在活動當中更熟悉高中生活，認識更多學長姊。

同時又可以學習更多關於 Python、C++、Tkinter、網頁設計的知識，還會參觀資訊產業的公司，也當然會讓大家遊玩各式各樣好玩的神秘活動，期待你們的參與！

INFOR

建中資訊

INFOR 37th

建中資訊社是一個以電腦科技為主題的社團，歡迎對程式設計、網頁製作、遊戲開發、人工智慧等領域有興趣的同學加入！

活動介紹

豐富有趣的營隊活動，等你來體驗！

Day 1

大地遊戲

在寬廣的場地中進行小隊對抗，透過各種關卡與趣味競賽，拉近團員們的距離並爭取最高榮譽！

Day 2

企業參訪

造訪知名資訊科技公司，深入了解業界運作，並邀請工程師學長姊分享寶貴的工作與學習經驗。

Day 3

RPG 角色扮演遊戲

真人實體 RPG！燒腦解謎與劇情推進，考驗你的觀察力、邏輯推理以及團隊默契！

Day 4

大型團康活動

走出戶外探索雙北，結合各類創意互動任務，一起創造熱血又難忘的回憶！

Day 5

桌遊與趣味競賽

考驗智慧與運氣的趣味桌遊與團體遊戲，看哪個小隊能成為最後的贏家！

Day 6

神秘活動

營隊的壓軸重頭戲，最具氛圍感與情感連結的難忘之夜，敬請期待！

課程介紹

我們有許多不同的資訊相關課程，一起來認識這些課程吧！

網頁前端

負責人：陳柏祐

本課程主要介紹網站介面設計與互動開發的基礎知識。學習 HTML、CSS 與 JavaScript 的核心概念，了解網頁結構、版面配置與動態效果的製作方式。透過實作練習，設計出美觀且具互動性的網頁。

  • 網頁結構介紹
  • HTML
  • CSS
  • JavaScript

Discord.py

負責人：鄭紹禹

本課程使用 Python 以及 discord.py 函式庫進行 DC Bot 教學開發，學習 discord.py 使用的非同步語法 (async & await) ，從基礎 (建置機器人取得 Token, 設置 Python 環境) 一步步到實作 (連結外部 API, 進階功能)。

  • 非同步 Python 語法 (async, await)
  • 環境建置 (創建 DC Bot, discord.py 安裝)
  • 基礎機器人功能 (偵測訊息、發送訊息)
  • 進階機器人功能 (連結外部 API, 使用 Embed 以及其他元件製作可互動的簡易使用者介面)

C++

負責人：賴語穗

這是一個基礎的 C++ 課程！在這裡，我們將讓你掌握核心觀念與實用技巧，我們將深入淺出的認識以下主題，讓你對 C++ 語法有更基本的認識。

  • 編寫 C++ 的電腦環境配置
  • 輸入／輸出
  • 各種變數型態與應用
  • if-else 條件判斷
  • 運算子
  • for / while 迴圈

Python

負責人：王裕佳

Python 是一種語法簡潔、易於閱讀且受歡迎的程式語言，非常適合初學者。你將掌握其核心概念和實用技巧，我們將深入淺出地學習以下主題，讓你能夠建立堅實的程式設計基礎。還有更多有趣的內容等你來學習！

  • 環境認識與準備
  • 各種變數型態與其應用
  • 運算
  • 條件
  • 迴圈

Python turtle

負責人：林羿秀

Welcome to the world of Python Turtle! 歡迎來到海龜繪圖的奇幻世界！在這堂課中，我們將透過最直觀的圖形化反饋，帶領你一手掌握 Python 的程式核心邏輯。你將透過指揮小海龜在螢幕上爬行，創作出屬於你的數位幾何藝術品！本課程將循序漸進，從基礎線條到靈活運用迴圈創作璀璨星空與萬花筒圖案，帶領大家把腦中的想像具象化為現實！

  • turtle 基礎控制：畫布初始化、海龜前後移動、旋轉與座標系統概念
  • 色彩與畫筆藝術：調整畫筆粗細、自訂色彩、圖形填色技巧
  • 結構化你的畫作：靈活運用 for 迴圈繪製多邊形與幾何圖形
  • 專案實作：生成你想要的圖形
